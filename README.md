# javascript_lesbases

A simple repository with basic JavaScript exercises and examples. Each file contains a small script that demonstrates a fundamental programming concept, from user interaction and loops to array manipulation and string processing.

## Contents

- `script_1.js` - Basic output and greeting. Displays a welcome message and asks the user for their name.
- `script_2.js` - Factorial calculation. Defines a function to compute the factorial of a number and prints the result.
- `script_3.js` - Pyramid drawing. Builds and logs a text pyramid using loops and string repetition.
- `script_4.js` - Entrepreneur data processing. Uses arrays of objects to filter, map, compute age, and sort values.
- `script_5.js` - Library book analysis. Evaluates book rental data, finds the most and least rented books, and sorts the remaining list.
- `script_6.js` - RNA translation. Translates nucleotide codons into amino acid names with a custom mapping.
- `script_7.js` - Chatbot interaction. Responds to user input with simple rules for questions, uppercase text, and keyword matching.

## How to use

1. Open the project folder in a JavaScript environment.
2. Run a script with Node.js or in a browser console (for scripts using `prompt`).
3. Review the code to understand basic JavaScript techniques.

## Purpose

This folder is intended as a beginner-friendly collection of JavaScript examples for learning and practicing core programming concepts.
